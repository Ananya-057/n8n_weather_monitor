# 🌦️ n8n Weather Monitoring Bot

An automated weather monitoring workflow built using n8n that fetches real-time weather data and sends updates to a Telegram bot.

## Features
- 🌡️ Real-time temperature monitoring
- 💧 Humidity tracking
- 🌬️ Wind speed updates
- ☀️ Weather condition alerts
- 🤖 Automatic Telegram notifications

## Tech Stack
- n8n
- JavaScript
- HTTP Request
- Telegram Bot API
- Weather API

## Workflow

Manual Trigger
↓
HTTP Request
↓
JavaScript Processing
↓
Telegram Notification
